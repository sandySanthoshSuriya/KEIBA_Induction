To Listen
  1. Establish the Sophos VPN Connection.

  2. Follow the steps to setup the proxy for different browsers.

  3. Establish the VM ware Horizon client Connection.

  4. Setup should be made in remote system.
Proxy Setup

Step 1 : Add the proxy switchy omega extension in chrome by using the below link.
https://chromewebstore.google.com/detail/proxy-switchyomega-3- zero/pfnededegaaopdmhkdmcofjmoldfiped?hl=en

Step 2 : Click New profile in the left menu.

Step 3 : Enter "Profile Name depending on Environment and select "PAC Profile". Then, click "Create" button.

Step 4 : Currently we are using 2 Pac url for on-site and off-site.
On-site pac : http://stg-keiba.r10s.jp/proxy/stg3.pac
Off-site pac : https://keiba.rakuten.co.jp/proxy/stg3.pac

Step 5 : Click the Download Profile Now button.


Step 6 : Click Apply changes button.

Download the certificate

Step 1 : Click the below link and download the file.

RDEV : http://stg-m.keiba.rakuten.co.jp/proxy/rdev.der

STG.CHECKPROXY : http://stg-m.keiba.rakuten.co.jp/proxy/stg.check.crt

Step 2 : Select manage certificate in privacy and security

Step 3 : Click Trusted Rooot certificate Authority and import the certificate and click OK button

To Listen
Proxy Setup

Step 1 : In Windows settings, go to Proxy Setting.

Step 2 : Ensure that “Automatically detect settings and Use setup script” toggle buttons are enabled.

Step 3 : Enter the Staging URL Ex :
On-site pac : http://stg-keiba.r10s.jp/proxy/stg3.pac
Off-site pac:https://keiba.rakuten.co.jp/proxy/stg3.pac in Script Address and click Save.

Step 4 : Staging setup is ready now for Microsoft Edge.

To Listen
Proxy Setup

Step 1 : Click Preferences from Finder at the top left.

Step 2 : Click the Network icon.

Step 3 : Click Advanced settings.

Step 4 : Click Proxies and select Automatic Proxy Configuration.
Enter the staging URL in Proxy Configuration file URL field.

Ex :
On-site pac : http://stg-keiba.r10s.jp/proxy/new-stg3.pac
Off-site pac:http://stg-keiba.r10s.jp/proxy/stg2.pac

Step 5 : Click Ok and Apply button.

Widget Creation

To create widget(Based on device the steps may slightly vary but there is no impact)

Step 1 : Long press any empty space from mobile home screen and click Widget option

Step 2 : Long press and drag the Rakuten Keiba App Widget to home screen

Step 3 : Rakuten Keiba widget will added on the home screen

Step 4 : In the widget, Today's Sale information, refresh button and current time will be displayed

Step 5 : Long press and drag the Rakuten Keiba App Widget to home screen.

Widget Size Change

Widget size can be changed only for Android devices not for iOS devices
It will be changed by Hold and draging on the widget
Widget can be set into Large, Medium and small

Large Width Medium Width Small Width
Display 3D widget

Step 1 : Long press the application for 3D Widget

The below details will be displayed in 3D widget
1. 投票
2. 入金・精算
3. オッズ
4. ライブ映像

Widget Creation

To create widget(Based on device the steps may slightly vary but there is no impact)

Step 1 : Swipe left from the mobile home screen and click edit option

Step 2 : Click customize

Step 3 : Click add icon on the Rakuten Keiba App

Step 4 : Click Done

Step 5 : Click Done Rakuten Keiba App widget will be added. In the widget, Today's Sale information will be displayed.

Display 3D widget

Step 1 : Long press the application for 3D Widget

The below details will be displayed in 3D widget
1. 投票
2. 入金・精算
3. オッズ
4. ライブ映像

Pitari Banner

Pitari banner is an MNO marketing on Mobile APPs.
MNO would like to display banner to Rakuten mobile applications to acquire new users.
Pitari banner is displayed at Top of the Home Screen for all three users and not login state.

Precondition: In Firebase the pitari banner status should be set as TRUE to display the banner in Home tab Top of the screen.

Note : In Firebase Application the pitari banner status is set as FALSE the banner will not be displayed

Step 1 : Open the keiba App, the pitari banner is displayed at TOP of the Home screen

Step 2 : On clicking the Pitari banner the screen navigates to the external browser displaying the banner information page.

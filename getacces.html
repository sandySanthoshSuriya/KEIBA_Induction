JIRA Access

Step 1 : Please Access the DLM Link
Link : DLM

Step 2 : After Navigate the link please select option for tech-sqag-jira

Step 3 : After select that option and scroll down click Add button

Step 4 : After click the Add button pop up will show, please add Rakuten mail Address.
Example : os-poorn.a.selvaraju@rakuten.com

Step 5 : Please click ok button and Added that rakuten mail Address, After added please inform to onsite manager to approve the DLM group.

To Listen
RDEV VPN [For Mobile] Access

Step 1 : To access the RDEV link
URL : RDEV

Step 2 : After navigate the link please check your os user name and click Send request for approval button
Example : os-gokul.karunakaran

Step 3 : After click that button automatically create jira ticket

Step 4 : After created the ticket please wait for approval message. Once we got the approval we get access for vpn
Note : If not approval with in 7 days the ticket closed automatically. Again we need to create new ticket for VPN access.
